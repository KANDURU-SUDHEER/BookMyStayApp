

This project is the initial setup for a **Hotel Booking Management System** developed in Java. The application begins execution from the `BookMyStayApp` class, which serves as the main entry point for the program. In this first use case, the system simply launches through the `main()` method and displays a welcome message to confirm that the application has started correctly. The purpose of this stage is to establish a clear startup structure for the program and verify that the development environment and Java runtime are functioning properly.

At this stage, the application does not yet include business logic, data handling, or user interaction. It is designed only to demonstrate successful system initialization and provide a foundation for future development. In later use cases, additional features such as hotel listings, booking functionality, user input handling, and data management will be implemented to gradually build the complete Hotel Booking Management System.
